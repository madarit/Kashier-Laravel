# Changelog

All notable changes to `laravel-kashier` will be documented in this file.

## 1.0.0 - 2025-11-24

### Added
- Initial release
- Kashier payment gateway integration
- iFrame payment support
- Hosted Payment Page (HPP) support
- Automatic signature validation
- Test and live mode configuration
- Facade support for easy access
- Customizable views
- Auto-discovery for Laravel 5.5+
